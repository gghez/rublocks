About myblog

A running example for the rublocks language.

What is this?

myblog is the canonical playground for rublocks — a declarative JSON language that compiles to a Rust/Axum web application. Every page you see here is the output of files under playground/: models, routes, templates, and a layout, with the request pipeline expressed as a composition of named blocks.

What blocks are at work?

  • db.find_many — powers the homepage feed, the authors index, and the API list endpoints.
  • db.find_one — loads a single post, author, or draft preview; pairs with error via on_missing for 404s.
  • db.insert — persists a new comment from the post detail form.
  • guard — authorizes the /admin/posts/:slug/preview route against a CEL predicate on the ?token= query parameter.
  • error — short-circuits the handler with a typed HTTP error and a machine-readable code.
  • time.now — runs in the layout to bind the current year for the footer.

Try the API

  • GET /api/posts — list published posts.
  • GET /api/posts/:slug — one post with its author.
  • GET /api/authors — list authors.
  • GET /api/authors/:id — one author.
